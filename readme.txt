== Description ==

Primary Category extends default WordPress functionality to include the ability to promote posts/custom post type categories to be designated as a "Primary".  Easily display results on the frontend with [primary-posts] shortcode provided.

Major features in Primary Category include:

* Compatible with Classic Editor & Gutenberg
* Ability to assign a "primary" category on posts/custom post types
* Ability to remove "primary" category on posts/custom post types
* Enforce security (users must have manage_categories capability)
* Shortcode [primary-posts] to easily output primary posts (filtered by post type)

== Installation ==

Upload the Primary Category plugin to your blog, then activate it.
